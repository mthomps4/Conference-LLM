JARVIS is an internal AI agent management tool. Key product goals (as of 2026-03-30):

1. **Agent status at a glance** — see which project agents/teams are active, idle, waiting, errored
2. **Conversations awaiting input** — quick filter/nav to threads that need the user's attention
3. **Refined permissions** — per-persona per-conversation file/tool access controls
4. **Slack-like sidebar** — compact thread names only, details inside the thread view
5. **Project organization** — group conversations by project, see project-level status

**Why:** Matt is a company owner managing multiple projects via AI agents. Needs the same kind of oversight a manager gets from Slack/project tools, but for AI conversations.
